# 5.5 Langetermijnpositionering

VeriSeal is niet:

- Een workflowtool,
- Een documentbeheersysteem,
- Een berichtenkanaal,
- Een opslagprovider.

Het is:

Een integriteitssubstraat.

- Een commercieel inzetbare integriteitsinfrastructuur
