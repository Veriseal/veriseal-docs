# Dreigingsmodel & Adversariële Analyse

Deze bijlage biedt een gestructureerde adversariële analyse van de VeriSeal integriteitsinfrastructuur.

## Reikwijdte

Dit document definieert:

- Dreigingsactoren
- Vertrouwensgrenzen
- Aanvalsoppervlakken
- Mitigaties
- Restanten van risico's
- Governancevereisten

VeriSeal beveiligt structurele integriteit, niet semantische waarheid.
