# Beveiligingsveronderstellingen

VeriSeal vertrouwt op de volgende veronderstellingen:

- SHA-256 blijft botsingsbestendig
- Handtekeningschema's blijven veilig
- Canonieke serialisatie is deterministisch
- Privésleutels worden beschermd onder institutioneel bestuur
- Alleen-bijvoegbeleid voor grootboek wordt operationeel gehandhaafd

Beveiliging is cryptografisch, maar veerkracht is afhankelijk van bestuur.
