# Bewijsinfrastructuur

## Bewijskern
Hashing, Merkle-wortels, alleen-toevoegbare grootboek, deterministische openbare JSON.

## Scheiding van lagen
Backend vs openbare verificateur UI vs PDF-renderer (alleen renderen).

## Sterke binding
Bind UX-log + media + grootboek + JSON (+ PDF/OTS indien ingeschakeld).

## Operaties
Soevereine implementatie, versterkte TLS, gecontroleerde sleutels, geplande OTS-automatisering.
