# Cryptografische Grondslagen

VeriSeal vertrouwt op gevestigde cryptografische primitieve:
- hashing (bijv. SHA-256),
- asymmetrische handtekeningen voor attestaties,
- optionele tijdstempelverankering (bijv. blockchain/OpenTimestamps).

Het doel is niet nieuwigheid. Het doel is determinisme.

Een bewijs moet zijn:
- reproduceerbaar,
- verifieerbaar door derden,
- stabiel in de tijd.

Cryptografie biedt wiskundige integriteit, geen semantische waarheid.
