# Supuestos de Seguridad

VeriSeal se basa en los siguientes supuestos:

- SHA-256 sigue siendo resistente a colisiones
- Los esquemas de firma permanecen seguros
- La serialización canónica es determinista
- Las claves privadas están protegidas bajo gobernanza institucional
- La política de libro mayor de solo anexar se aplica operativamente

La seguridad es criptográfica, pero la resiliencia depende de la gobernanza.
