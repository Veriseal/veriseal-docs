# Fundamentos Criptográficos

VeriSeal se basa en primitivas criptográficas establecidas:
- hashing (por ejemplo, SHA-256),
- firmas asimétricas para atestaciones,
- anclaje de marca de tiempo opcional (por ejemplo, blockchain/OpenTimestamps).

El objetivo no es la novedad. El objetivo es el determinismo.

Una prueba debe ser:
- reproducible,
- verificable por terceros,
- estable en el tiempo.

La criptografía proporciona integridad matemática, no verdad semántica.
