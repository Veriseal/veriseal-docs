# Infraestructura de Evidencia

## Núcleo de prueba
Hashing, raíces de Merkle, libro mayor de solo anexar, JSON público determinista.

## Separación de capas
Backend vs interfaz de verificación pública vs renderizador de PDF (solo renderizado).

## Vinculación fuerte
Vincular registro de UX + medios + libro mayor + JSON (+ PDF/OTS cuando esté habilitado).

## Operaciones
Despliegue soberano, TLS reforzado, claves controladas, automatización programada de OTS.
