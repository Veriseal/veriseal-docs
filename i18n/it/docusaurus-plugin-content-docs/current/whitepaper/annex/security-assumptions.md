# Assunzioni di Sicurezza

VeriSeal si basa sulle seguenti assunzioni:

- SHA-256 rimane resistente alle collisioni
- Gli schemi di firma rimangono sicuri
- La serializzazione canonica è deterministica
- Le chiavi private sono protette sotto la governance istituzionale
- La politica del registro solo aggiuntivo è applicata operativamente

La sicurezza è crittografica, ma la resilienza dipende dalla governance.
