# Infrastruttura delle Prove

## Nucleo di prova
Hashing, radici Merkle, registro solo aggiuntivo, JSON pubblico deterministico.

## Separazione dei livelli
Backend vs interfaccia utente del verificatore pubblico vs renderer PDF (solo rendering).

## Vincolo forte
Vincolare log UX + media + registro + JSON (+ PDF/OTS quando abilitato).

## Operazioni
Distribuzione sovrana, TLS rinforzato, chiavi controllate, automazione OTS programmata.
