# Fondamenti Criptografici

VeriSeal si basa su primitive crittografiche consolidate:
- hashing (ad es., SHA-256),
- firme asimmetriche per attestazioni,
- ancoraggio temporale opzionale (ad es., blockchain/OpenTimestamps).

L'obiettivo non è la novità. L'obiettivo è il determinismo.

Una prova deve essere:
- riproducibile,
- verificabile da terze parti,
- stabile nel tempo.

La crittografia fornisce integrità matematica, non verità semantica.
