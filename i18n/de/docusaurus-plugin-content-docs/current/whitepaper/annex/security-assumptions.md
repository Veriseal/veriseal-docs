# Sicherheitsannahmen

VeriSeal stützt sich auf die folgenden Annahmen:

- SHA-256 bleibt kollisionsresistent
- Signaturschemata bleiben sicher
- Kanonische Serialisierung ist deterministisch
- Private Schlüssel sind unter institutioneller Verwaltung geschützt
- Die Nur-Anfüge-Politik des Ledgers wird operationell durchgesetzt

Sicherheit ist kryptographisch, aber die Widerstandsfähigkeit hängt von der Governance ab.
