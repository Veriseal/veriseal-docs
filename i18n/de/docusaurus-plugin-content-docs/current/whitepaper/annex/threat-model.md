# Bedrohungsmodell & Gegnerische Analyse

Dieser Anhang bietet eine strukturierte gegnerische Analyse der VeriSeal-Integritätsinfrastruktur.

## Umfang

Dieses Dokument definiert:

- Bedrohungsakteure
- Vertrauensgrenzen
- Angriffsflächen
- Gegenmaßnahmen
- Restrisiken
- Governance-Anforderungen

VeriSeal sichert die strukturelle Integrität, nicht die semantische Wahrheit.
