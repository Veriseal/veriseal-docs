# Nachweis-Infrastruktur

## Beweiskern
Hashing, Merkle-Wurzeln, nur-anhängbares Hauptbuch, deterministisches öffentliches JSON.

## Trennung der Schichten
Backend vs. öffentliches Verifizierungs-UI vs. PDF-Renderer (nur-Render).

## Starke Bindung
Binden von UX-Log + Medien + Hauptbuch + JSON (+ PDF/OTS, wenn aktiviert).

## Operationen
Souveräne Bereitstellung, gehärtetes TLS, kontrollierte Schlüssel, geplante OTS-Automatisierung.
