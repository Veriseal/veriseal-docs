# Kryptographische Grundlagen

VeriSeal stützt sich auf etablierte kryptographische Primitive:
- Hashing (z.B. SHA-256),
- asymmetrische Signaturen für Beglaubigungen,
- optionale Zeitstempelverankerung (z.B. Blockchain/OpenTimestamps).

Das Ziel ist nicht Neuheit. Das Ziel ist Determinismus.

Ein Nachweis muss:
- reproduzierbar,
- von Dritten verifizierbar,
- über die Zeit stabil sein.

Kryptographie bietet mathematische Integrität, nicht semantische Wahrheit.
